# Upload Checklist

This is the flat browser-upload edition.

1. Create public repo: `axezent-ai-ctr-kernel`.
2. Upload every visible file from this folder.
3. If `.github` uploads, Actions will run automatically.
4. If `.github` does not upload, create `.github/workflows/main.yml` and paste `GITHUB_WORKFLOW_MAIN.yml`.
5. Commit with: `Launch Axezent AI CTR Kernel v1.0.0 flat browser upload`.
